# Shadcn Demo

## Get Started 

```bash
npx create-next-app@latest shadcn-demo --typescript --tailwind --eslint
cd shadcn-demo
npx shadcn-ui@latest init
npm run dev
```

## Add Components

```bash
npx shadcn-ui@latest add <component>
```
